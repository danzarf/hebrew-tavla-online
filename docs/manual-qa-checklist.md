# רשימת QA ידנית - ששבש טורקי אונליין

מסמך זה משמש כרשימת בדיקות חובה לפני ואחרי כל שינוי בפרויקט. המטרה היא לוודא שהמשחק הקיים לא נשבר, במיוחד סביב החוקים המיוחדים והבאגים שחזרו בעבר.

## כללי בדיקה

- לבדוק בדפדפן רגיל לפני כל שינוי משמעותי.
- לבדוק שוב אחרי כל שינוי.
- באונליין לבדוק עם שני חלונות, שני טאבים או שני מכשירים.
- לא לאשר שינוי אם אחד מהבאגים החוזרים מופיע שוב.
- אם שינוי נוגע ללוגיקה, לבדוק גם משחק מול מחשב וגם אונליין.
- אם שינוי נוגע ל-UI, לבדוק גם מסך דסקטופ וגם מסך מובייל צר.

## בדיקת טעינה בסיסית

- [ ] המשחק נטען בלי שגיאות בקונסול.
- [ ] הלובי נפתח בעברית ובכיוון RTL.
- [ ] קיימות אפשרויות משחק מול מחשב, משחק מול חבר וחוקים.
- [ ] הלוח מוצג בצורה תקינה.
- [ ] מגשי `White off` ו-`Black off` נמצאים מחוץ ללוח בצד שמאל.
- [ ] פאנל החוקים ופאנל המידע נכנסים למסך.
- [ ] כפתורי גלגול, Undo, End Turn ומשחק חדש מוצגים.

## משחק מול מחשב

- [ ] אפשר להתחיל משחק מול מחשב.
- [ ] גלגולת פתיחה מתבצעת.
- [ ] אם יש תיקו בגלגולת הפתיחה, מגלגלים שוב.
- [ ] מי שמנצח בגלגולת הפתיחה מתחיל ומשחק בלבן.
- [ ] תור השחקן מאפשר גלגול.
- [ ] תור המחשב מתבצע אוטומטית.
- [ ] המחשב מזיז אבנים עם אנימציה.
- [ ] המשחק לא נתקע אחרי כמה תורות.
- [ ] לוג המהלכים מתעד גלגולים, תנועות, אכילה והוצאה.

## בדיקת תנועה רגילה

- [ ] אפשר לבחור אבן חוקית.
- [ ] מוצגים יעדים חוקיים.
- [ ] לחיצה על יעד חוקי מזיזה את האבן.
- [ ] אכילה מעבירה אבן יריבה לבר.
- [ ] אם יש אבן על הבר, חייבים להכניס אותה לפני הזזת אבנים אחרות.
- [ ] אי אפשר להיכנס לנקודה חסומה.
- [ ] הוצאה מהלוח אפשרית רק כשכל האבנים בבית.
- [ ] Pip count מתעדכן אחרי תנועה.

## Undo

- [ ] אחרי מהלך אחד בתוך תור, כפתור Undo נשאר זמין.
- [ ] Undo מחזיר את הלוח למצב הקודם.
- [ ] Undo מחזיר גם remaining moves, dice, bar ו-off.
- [ ] Undo לא נעלם עד שכל התור מסתיים או עד שהתור נגמר רשמית.
- [ ] אחרי סוף תור, Undo מושבת.
- [ ] Undo לא זמין בזמן תור היריב או תור המחשב.
- [ ] Undo לא גורם לסנכרון אונליין שגוי.

## End Turn

- [ ] אי אפשר ללחוץ End Turn כשעדיין יש מהלכים חוקיים זמינים.
- [ ] אפשר ללחוץ End Turn רק כשכל המהלכים נוצלו.
- [ ] אפשר ללחוץ End Turn כשאין מהלכים חוקיים שנותרו.
- [ ] אם יש מהלך חוקי אחד לפחות, End Turn נשאר מושבת.
- [ ] אחרי End Turn התור עובר לשחקן הנכון.
- [ ] בדאבל רגיל, סוף ארבעת המהלכים נותן תור נוסף לאותו שחקן.
- [ ] ב-`4-5`, סוף ארבעת המהלכים לא נותן תור נוסף.

## חוק `1-2` / `2-1`

- [ ] כאשר יוצא `1-2`, השחקן לא מזיז אבנים.
- [ ] כאשר יוצא `2-1`, השחקן לא מזיז אבנים.
- [ ] התור עובר מיד ליריב.
- [ ] לוג המשחק מציין שהתור אבד.
- [ ] Undo לא זמין כי לא בוצע מהלך.

## דאבל רגיל

- [ ] דאבל רגיל נותן ארבעה מהלכים.
- [ ] ארבעת המהלכים משתמשים בערך הדאבל.
- [ ] לאחר ארבעת המהלכים השחקן מקבל תור נוסף.
- [ ] אם אין מהלכים חוקיים, התור מסתיים לפי החוקים.
- [ ] Pip count והלוג מתעדכנים נכון.

## שלושה דאבלים ברצף

- [ ] אחרי דאבל ראשון, המשחק ממשיך.
- [ ] אחרי דאבל שני ברצף לאותו שחקן, המשחק ממשיך.
- [ ] אחרי דאבל שלישי ברצף, הצדדים/צבעים מתחלפים.
- [ ] השחקן ממשיך לשחק עם הצד החדש שלו.
- [ ] התור עובר בצורה תקינה אחרי ההחלפה.
- [ ] המשחק לא קופא.
- [ ] כפתורי Roll, Undo ו-End Turn חוזרים למצב תקין.
- [ ] באונליין שני הצדדים רואים את ההחלפה.

## חוק `4-5` / `5-4`

- [ ] כאשר יוצא `4-5`, נפתח חלון בחירת דאבל.
- [ ] כאשר יוצא `5-4`, נפתח חלון בחירת דאבל.
- [ ] אפשר לבחור כל דאבל מ-`1-1` עד `6-6`.
- [ ] הבחירה נותנת ארבעה מהלכים לפי הדאבל שנבחר.
- [ ] הדאבל הנבחר לא נותן תור נוסף.
- [ ] באונליין, הבחירה מסתנכרנת לשני השחקנים.
- [ ] החלון לא חוסם את הלוח אחרי הבחירה.

## חוק `5-6` / `6-5`

- [ ] כאשר יוצא `5-6`, השחקן מקבל שני מהלכים חופשיים.
- [ ] כאשר יוצא `6-5`, השחקן מקבל שני מהלכים חופשיים.
- [ ] אפשר לבצע שני מהלכים חוקיים לכל נקודה מותרת.
- [ ] אי אפשר לנחות על נקודה חסומה.
- [ ] אם אין אבנים על הבר, אפשר להזיז אבנים רגילות באופן חופשי.
- [ ] אם יש אבנים על הבר, חייבים להכניס מהבר קודם.
- [ ] אבני בר נכנסות רק לנקודות הכניסה הרגילות בבית היריב.
- [ ] אסור להציב אבן מהבר בכל מקום בלוח.
- [ ] אחרי שני מהלכים חופשיים התור מסתיים כראוי.

## ירושלמי

- [ ] כאשר שחקן מוציא 15 אבנים, המשחק לא מסתיים מיד בלי ירושלמי.
- [ ] המפסיד מקבל אפשרות לבחור שני מספרים.
- [ ] הבחירה מוצגת או מסונכרנת כראוי.
- [ ] המנצח מגלגל קוביות עם אנימציה.
- [ ] אם הגלגול מתאים בדיוק לבחירה, המפסיד גונב את הניצחון.
- [ ] אם הגלגול לא מתאים, המנצח המקורי מנצח.
- [ ] שני השחקנים רואים הצלחה או כישלון.
- [ ] שני השחקנים רואים את הודעת המנצח הסופית.
- [ ] המשחק לא מאפשר המשך תורות אחרי סיום ירושלמי.

- [ ] שני הצדדים רואים מי התחיל ומי משחק באיזה צבע.
