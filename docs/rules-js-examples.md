# דוגמאות שימוש ב-rules.js

מסמך זה מסביר בפשטות את הקובץ `src/game/rules.js`.

הקובץ הזה הוא התחלה קטנה ובטוחה של הפרדת חוקי משחק לקבצים נפרדים. כרגע הוא לא מחובר ל-`index.html`, ולכן הוא לא משנה את המשחק הקיים ולא משנה את ההתנהגות של המשחק ב-Vercel.

## מה יש בקובץ

`src/game/rules.js` כולל כמה פונקציות עזר קטנות:

- `WHITE` - הצבע הלבן.
- `BLACK` - הצבע השחור.
- `SIGN` - הדרך שבה הלוח מזהה אבנים לבנות ושחורות.
- `otherColor` - מחזירה את הצבע השני.
- `countColorAt` - סופרת כמה אבנים של צבע מסוים נמצאות בנקודה בלוח.
- `oppCountAt` - סופרת כמה אבנים של היריב נמצאות בנקודה בלוח.
- `isBlocked` - בודקת אם נקודה חסומה בגלל שיש עליה שתי אבני יריב או יותר.

## איך מיוצגים לבן ושחור

בקובץ הזה:

```js
WHITE = 'white'
BLACK = 'black'
```

בלוח עצמו, לפי הקוד הקיים:

- אבנים לבנות נשמרות כמספרים שליליים.
- אבנים שחורות נשמרות כמספרים חיוביים.
- נקודה ריקה היא `0`.

לדוגמה:

```js
const state = {
  board: [
    0,
    -2, // נקודה 1: שתי אבנים לבנות
    0,
    3,  // נקודה 3: שלוש אבנים שחורות
  ],
};
```

חשוב: בלוח המשחק הנקודות מתחילות מ-`1`, ולכן `board[1]` הוא נקודה 1.

## דוגמה: otherColor

```js
otherColor('white'); // מחזיר 'black'
otherColor('black'); // מחזיר 'white'
```

המטרה: לדעת מי היריב של הצבע הנוכחי.

## דוגמה: countColorAt

```js
const state = {
  board: [
    0,
    -2,
    0,
    3,
  ],
};

countColorAt(state, 1, 'white'); // מחזיר 2
countColorAt(state, 1, 'black'); // מחזיר 0
countColorAt(state, 3, 'black'); // מחזיר 3
countColorAt(state, 3, 'white'); // מחזיר 0
```

המטרה: לספור רק את האבנים של הצבע שביקשנו.

## דוגמה: oppCountAt

```js
const state = {
  board: [
    0,
    -2,
    0,
    3,
  ],
};

oppCountAt(state, 1, 'black'); // מחזיר 2, כי היריב של שחור הוא לבן
oppCountAt(state, 3, 'white'); // מחזיר 3, כי היריב של לבן הוא שחור
```

המטרה: לבדוק כמה אבני יריב נמצאות בנקודה מסוימת.

## דוגמה: isBlocked

נקודה חסומה אם יש עליה שתי אבנים או יותר של היריב.

```js
const state = {
  board: [
    0,
    -2,
    1,
    3,
  ],
};

isBlocked(state, 1, 'black'); // מחזיר true, כי יש שם שתי אבנים לבנות
isBlocked(state, 2, 'white'); // מחזיר false, כי יש שם רק אבן שחורה אחת
isBlocked(state, 3, 'white'); // מחזיר true, כי יש שם שלוש אבנים שחורות
```

המטרה: לדעת אם שחקן יכול להיכנס לנקודה או שהיא חסומה.

## למה זה בטוח

הקובץ `src/game/rules.js` עדיין לא מיובא ולא מופעל מתוך `index.html`.

לכן:

- אין שינוי במשחק הקיים.
- אין שינוי ב-Firebase.
- אין שינוי ב-Vercel.
- אין שינוי באנימציות, UI, Undo, End Turn, אונליין או ירושלמי.

זה רק תיעוד ודוגמאות כדי שנוכל להמשיך בהדרגה ובזהירות.
